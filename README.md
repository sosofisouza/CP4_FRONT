# CP4_FRONT
🎵 MUSIC
Landing Page Responsiva
Checkpoint 4 — Front-End Design

================================================================

SOBRE O PROJETO
----------------------------------------------------------------
Music é uma landing page desenvolvida como projeto de Front-End,
com foco em design responsivo, identidade visual consistente e
boa experiência do usuário em diferentes dispositivos. A página
apresenta uma plataforma fictícia de streaming musical com layout
moderno e adaptável.

================================================================

TECNOLOGIAS UTILIZADAS
----------------------------------------------------------------
• HTML5          — Estrutura semântica da página
• CSS3           — Estilização, animações e transições
• Flexbox & Grid — Organização e alinhamento dos elementos
• Media Queries  — Responsividade para dispositivos móveis
• Google Fonts   — Tipografia com a fonte Quicksand
• Font Awesome   — Ícones dos botões e elementos visuais
• Boxicons       — Ícones das redes sociais no rodapé

================================================================

ESTRUTURA DE ARQUIVOS
----------------------------------------------------------------
• index.html                — Página principal
• assets/style.css          — Estilos globais e componentes
• assets/responsividade.css — Media queries para mobile
• assets/reset.css          — Reset de estilos padrão do navegador
• imagens/                  — Imagens e ilustrações do projeto

================================================================

FUNCIONALIDADES
----------------------------------------------------------------
• Layout totalmente responsivo para desktop e mobile
• Menu de navegação com botão de Login estilizado
• Seções Hero com imagem e texto em destaque
• Grade de artistas com imagens e hover interativo
• Rodapé com links organizados em colunas
• Ícones de redes sociais (Instagram, Twitter, Discord)
• Gradiente de fundo personalizado
• Animações e transições suaves nos elementos interativos

================================================================

COMO EXECUTAR
----------------------------------------------------------------
Não é necessária nenhuma instalação. Basta:

1. Fazer o download ou clonar este repositório
2. Abrir o arquivo index.html diretamente no navegador
